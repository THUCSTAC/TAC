# 编程作业的创新设计 
Innovative design of Programming Assignments

讨论问题：

- 考虑到大语言模型的广泛出现，程序设计基础的编程作业应该如何创新？
    - 曾经的程序设计基础作业形式是什么？
    - 大语言模型给曾经的作业形式带来了什么问题？
    - 程序设计基础归根结底要教给学生什么？
  
- 你最喜欢的课程作业（不一定是计算机专业的）是什么？（这个讨论活动效果不是特别好）
    - 这项作业有什么特点？
    - 能否借鉴到编程作业的创新设计当中？

- 你最喜欢的电子游戏是什么？（这个讨论活动效果不是特别好）
    - 对你来说，它的“爽点”在哪里？
    - 这种特性能否融入到编程作业的创新设计中？
